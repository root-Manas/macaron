# Launch Playbook

## Goal

Ship a repeatable launch package for each meaningful release.

## Assets

- 40-60 second CLI+UI demo GIF
- 5 screenshots (setup, scan, status, results, dashboard)
- release note with:
  - what changed
  - migration impact
  - benchmark snapshot

## Channels

1. X / Twitter launch thread
2. LinkedIn technical summary
3. Reddit post in allowed security tooling communities
4. GitHub release notes + pinned discussion

## Thread structure

1. One-line problem statement
2. 3 differentiators
3. 4 command quickstart
4. demo clip
5. link to release and docs

## Post-launch metrics (7-day)

- GitHub stars/week
- unique clone count
- issue/PR conversion
- release download count
